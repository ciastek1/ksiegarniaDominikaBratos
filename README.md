Wizytówka firmy Księgarnia "Słowo w słowo" | Dominika Bratos

Opis projektu:
Projekt przedstawia prostą stronę typu onepage, która jest wizytówką Księgarni "Słowo w słowo".
Strona zawiera sekcje: powitalną, opisu firmy, oferty oraz formularz kontaktowy.
Strona została wykonana za pomocą HTML, CSS oraz JavaScript.

Podczas wykonywania projektu nauczyłam się:
- dodawać proste interakcje w JavaScript
- korzystać z systemu kontroli wersji Git / tworzenia commitów (mniej więcej)

Największą trudnością było:
- dodawanie commitów (co widać)
- konfiguracja repozytorium GitHub
